== jMockit.example
Testbeispiele mit jMockit

=== Hinweise
jMockit arbeitet mit Instrumentierung des ByteCodes.

=== Erkenntnisse

* @Tested Objekte sind zunächst mal keine Mocks und ergo kann auch kein Mock Verhalten definiert werden.
   Evtl. lässt sich das lösen über

.Constructor.java
[source, java]
----
new MockUp<Familie>() {
       @Mock
       public void $init(Invocation invocation, String string) {
           ((Familie)invocation.getInvokedInstance()).sippenBezeichnung = string;
       }
   };
----
